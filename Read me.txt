CST-239 Milestone 4
Fantasy Game Store Program


Description

This project is a console-based Fantasy Game Store application developed in Java.



View products
Purchase products
View the shopping cart
Return purchased products
Exit the application

The store inventory includes weapons, armor, and health items.

For Milestone 4, the application was enhanced by adding file handling and inventory loading functionality. 
Product inventory data is stored in an inventory.json file and loaded through the FileService class. 
Custom exception handling was also implemented using the InventoryException class.

The application demonstrates object-oriented programming concepts including:

Inheritance
Encapsulation
Polymorphism
Collections
Exception Handling
File Processing
JSON Inventory Management
